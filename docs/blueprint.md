# **App Name**: PRODUCTION

## Core Features:

- Product Display: Showcase products with images, descriptions, and pricing.
- Mobile Responsiveness: Ensure the website is fully responsive and optimized for viewing on mobile devices.
- WhatsApp Integration: Implement a button for each product that, when clicked, opens WhatsApp with a pre-filled message containing the product details.
- Easy Navigation: Provide clear and intuitive navigation for users to browse products easily.
- Category Option: Allow products to be categorized.
- Category-wise Filter: Enable filtering of products based on their categories.
- Search Bar: Enable users to search for products using a search bar.

## Style Guidelines:

- Primary color: Deep Blue (#293462) to convey trust and reliability, suitable for e-commerce platforms.
- Background color: Light gray (#F0F4EF), desaturated, almost white.
- Accent color: Purple (#80485B), to make elements stand out without causing distraction.
- Body and headline font: 'Inter', a sans-serif font known for its clean, readable style, suitable for headlines and body text.
- Use simple, minimalist icons for navigation and product categories.
- Implement a clean, grid-based layout to ensure readability and a professional look.
- Use subtle animations (e.g., fading, sliding) on product images or button hover states to enhance user experience.
